In the lab 3, I conencted two VL53L1X Time-of-Flight Distance Sensors to the Artemis board. Then, I recorded the data from them and tested and discussed about the speed and distance of ToF sensor.

Shihming Lin
Feb 18, 2024

Part 1: Prelab

According to datasheet, the sensor's default I2C address is 0x52. However, the address is 0x29 as shown in Figure 1. This is the same as described in Pololu website . Since ToF is a slave device, it only has 7bit address (0x29 = 0101001). However, if an 8-bit representation is used with the read/write bits as least significant bits (0 for write and 1 for read), the 7-bit address (0x29) will be shifted one bit to the left and the read/write bit will be appended to that address so that the address will be 0x52 = 01010010.
